# FitLife Pro Assessment

Fitness & Wellness platform developed as part of the DOTMappers Web Development Assessment.

## Project Overview

FitLife Pro includes:

- Custom WordPress Theme
- Custom WordPress Plugin
- Gutenberg Blocks
- WooCommerce Store
- Shopify FitWear Storefront

## Tech Stack

### WordPress
- PHP
- WordPress Custom Theme Development
- Custom Plugin Development
- Gutenberg Blocks
- WooCommerce

### Shopify
- Liquid
- Shopify Sections
- Shopify Storefront API
- AJAX Cart API

## Installation

### WordPress Setup

1. Install WordPress
2. Copy theme into:

wp-content/themes/

3. Copy plugin into:

wp-content/plugins/

4. Activate:
- FitLife Pro Theme
- FitLife Core Plugin

## Requirements

- PHP 8+
- WordPress 6+
- WooCommerce
- Node.js (for Gutenberg blocks)

## Features

### WordPress
- Trainer CPT
